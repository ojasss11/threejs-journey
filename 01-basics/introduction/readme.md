### What is WebGL and Three.js?

**WebGL**  
It is a technology that lets the browser draw 3D graphics using the computer's graphics card (GPU).

**Three.js**  
It is a helpful library that makes WebGL much easier to use. Instead of writing complicated code, Three.js allows us to create 3D scenes in a simple way.
