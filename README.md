# threejs-journey 🚀

Learning 3D on the web — one project at a time.

This is my personal journey into Three.js and interactive 3D development.

I started from absolute zero with no prior experience. Every folder here shows real progress as I build small focused projects and improve step by step.

### What This Repo Is About

- Understanding how 3D graphics work in the browser
- Creating interactive and immersive experiences
- Learning scenes, cameras, animation, lighting, and rendering
- Growing through consistent hands-on practice

### Projects So Far

**01 — Basics**  
My first steps into Three.js.  
Setting up the scene, rendering my very first 3D object, and learning the core concepts.

**02 — First Scene**  
Bringing the scene to life.  
Object transformations, basic animations, working with cameras, and making everything responsive.
